📋 How to Use Sound Analysis

  1. Add TikTok Sound URLs: Paste URLs of TikTok sounds you want to analyze (one per line)
  2. Configure Settings: Set your Last.fm API key for genre analysis (optional)
  3. Click each sound link: Use the Sound Data Collector extension to collect creator usernames and hashtags
  4. Edit Song Names: Click the pencil icon to edit song titles for better genre matching
  5. Input Creator Lists: Return here and add the creators you've collected for each sound
  6. Run Analysis: Process the data to find creators who appear across multiple sounds, analyze hashtags, and fetch genres
  7. Export Results: Download CSV files for your marketing campaigns
